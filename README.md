
Графическое приложение для запуска pytest-тестов с удобным интерфейсом.  
Поддерживает тёмную тему, фильтрацию тестов, поиск, остановку выполнения и сохранение результатов.

 Возможности

-  **Автоматическое обнаружение тестов** – все файлы `test_*.py` в папке `tests/` подхватываются автоматически.
-  **Выборочный запуск** – отмечайте нужные тесты чекбоксами или запускайте один тест двойным кликом.
-  **Фильтрация по маркерам** – `slow`, `fast`, `integration`.
-  **Поиск по названию теста** – быстро находите нужные тесты.
-  **Остановка тестов** – прерывание выполнения в любой момент.
-  **Сохранение лога** – результат тестов можно сохранить в текстовый файл.
-  **Тёмная тема** – комфортная работа при слабом освещении.
-  **Сборка в один .exe** – готовое приложение для Windows без необходимости устанавливать Python.

 Требования

- Python 3.7+ (для запуска из исходников)
- Установленные библиотеки: `pytest` и `pytest-cov`

 Установка и запуск из исходников

1. Клонируйте репозиторий:
   
   git clone https://github.com/Bipolar123123/gui_test.git
   cd gui_test
   
2. Создайте виртуальное окружение:

python -m venv venv
venv\Scripts\activate      # Windows

3. Установите зависимости:

pip install -r requirements.txt

4. Запустите приложение:

python gui/test_runner.py

Сборка исполняемого файла (.exe)
Для Windows:

pip install pyinstaller
python -m PyInstaller --onefile --windowed --name "TestRunner" --add-data "tests;tests" --add-data "my_project;my_project" gui/test_runner.py

Как использовать

1. Выбор тестов – отметьте нужные чекбоксы или нажмите «Выбрать все».
2. Фильтрация – выберите маркер в выпадающем списке или введите текст в поле поиска.
3. Запуск – нажмите «Запустить выбранные тесты». Прогресс-бар покажет ход выполнения.
4. Остановка – если тесты идут слишком долго, нажмите «Остановить».
5. Просмотр результатов – вывод pytest отображается в цвете (зелёный – PASSED, красный – FAILED, синий – время выполнения).
6. Сохранение лога – нажмите «Сохранить лог» и выберите имя файла.
